# Mench-ai: A Blueprint for a Kosher AI Academic Tutor

## Overview

Mench-ai is a sophisticated AI tutoring platform designed to provide world-class academic education within a "kosher" digital environment. The platform operates as a high-level conversational AI tutor, comparable to leading models like Gemini 2.5, but with robust, values-centric architecture ensuring every interaction is safe, respectful, and aligned with Orthodox Jewish principles.

## Core Features

### 1. Dual-Mode AI System
- **Academic Tutor**: Ultra-Orthodox University Professor persona for Math, English, Science, and Computer Coding
- **Religious Tutor**: Council of Rabbinic Personas for religious questions and guidance

### 2. Kosher AI Architecture
- **Bi-directional Content Filtering**: Input validation and output screening
- **Topic Steering and Redirection**: Sophisticated redirection for gray-area topics
- **Robust Guardrails**: Multi-layered safety system maintaining Torah values

### 3. Interactive Features
- **Live Code Preview**: Real-time rendering of HTML/CSS/JavaScript
- **Code Export**: Save work directly to local computer
- **Natural Conversation**: Clean, minimalist interface for focused learning

## Getting Started

1. Clone the repository
2. Set up environment variables (see config/README.md)
3. Install dependencies for both backend and frontend
4. Run the development servers

## Technical Architecture

- **Core AI Model**: State-of-the-art LLM with Gemini 2.5 capabilities
- **LLM Gateway**: Specialized service enforcing safety and persona rules
- **Content Filters**: Keyword detection, classification models, rule-based systems
- **Persona Router**: Dynamic switching between academic and religious modes
